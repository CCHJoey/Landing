Achievements

Here is a brief summarize of my projects and my working experience.

Pintos Operating System Development and Adjustment

Designed and implemented critical system components for the Pintos operating system, including the kernel, CPU management, System Calls, and Management System. Additionally, I implemented customized scheduling technique for special needs

Recipe Generator

Developed a Recipe Generator Web Application designed to create customized recipes based on user preferences. This application leverages AWS infrastructure for robust scalability and reliability, enhancing the overall functionality. By integrating OpenAI technologies, it further enriches user experience, offering tailored suggestions that meet individual requirements

Transfer Learning based AI Research

Aimed to demonstrate how the transfer learning technique can bypass the need for re-training models while still achieving high performance. This research explores the efficiency and effectiveness of transfer learning in leveraging pre-trained models to solve new, related problems, potentially reducing both computational costs and time to deployment

Eco-sustainable Renting System

led a team to develope the 'Eco-Sustainable Renting System', a platform that enables environmentally conscious rentals during the COVID-19 pandemic. Utilizing blockchain technology, the system employs smart contracts to ensure transaction security and promote sustainability. This initiative helps reduce environmental impact by supporting a sharing economy and minimizing resource waste. This project has also help us winning the second price in National Cloud Application Competition

Research Assistant

Collaborated with professor on an innovative project that leveraged new database technologies and AI algorithms to enhance the diagnostic accuracy of Traditional Chinese Medicine. This initiative aimed to modernize and validate ancient practices through cutting-edge data analysis, significantly improving patient outcomes in holistic healthcare settings

Teaching Assistant

Collaborated with professor to analyze current cyber security challenges, define strategic frameworks, and explore emerging trends in the field. Together, we developed comprehensive educational materials for a graduate-level cyber security course. This initiative aims to equip students with the knowledge and skills necessary to address modern security threats effectively and to stay ahead in a rapidly evolving cyber landscape

Cyber attack prediction model

Built models that employs machine learning and sophisticated feature engineering to predict whether network connections originate from attackers or normal users. Focusing on extracting and analyzing key features such as domain names and other relevant connection features, which are crucial for building accurate predictive models to enhance our cybersecurity in server side

Movie recommendation system

Developed a movie recommendation system that enhances user experience by leveraging both content-based and user-based algorithms. This dual approach allows the system to deliver highly personalized movie suggestions by analyzing individual viewing preferences alongside movie characteristics. The integration of these methods ensures a richer, more accurate recommendation experience.

Data base management system for US patent

Led a team to develope a web-based database management system tailored to handle US patent data over a decade, featuring advanced visualization functions and data mining tools. This system not only streamlines the storage, retrieval, and analysis of patent information but also enhances user interaction through dynamic visualizations and robust data analytics capabilities, facilitating deeper insights and more effective decision-making

Rent room system analysis and design

Conducted a comprehensive analysis of an existing room rental system, identifying key issues and proposing a new system to address these challenges. For the new system, I developed detailed ER diagrams, fishbone diagrams, context diagrams, and data flow diagrams to ensure a robust design and streamlined operations. This project not only highlighted critical inefficiencies but also laid the groundwork for a more efficient and user-friendly system
